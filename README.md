# PPDA Platform

Educational platform for school management, communication, and academic monitoring.

## Tech Stack

### Frontend
- Angular

### Backend
- Node.js
- Express

### Database
- MongoDB (planned)

## Main Features
- Role-based authentication
- Student attendance tracking
- Academic notes and observations
- School events
- Gallery system
- Parent-student communication
